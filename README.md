# AI-Based-Movie-Recommendation-System
# 🎬 Movie Recommendation System

## 📌 Overview

This project is a **Movie Recommendation System** built using Python and Machine Learning. It suggests movies similar to a given input movie based on content features such as genres, keywords, and descriptions.

The system uses **content-based filtering** and cosine similarity to recommend relevant movies.

---

## 🎯 Features

* Recommend movies based on user input
* Uses content-based filtering
* Simple and fast recommendations
* Easy to understand and use

---

## 🧠 How It Works

1. Movie data is collected in a dataset
2. Important features like genres and overview are combined
3. Text data is converted into numerical form using **CountVectorizer**
4. **Cosine Similarity** is used to find similarity between movies
5. Top similar movies are recommended
